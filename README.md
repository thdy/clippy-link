# Clippy Link

ワンクリックでページのタイトルと URL をクリップボードにコピーする Chrome 拡張機能。

## 機能

- **ワンクリックコピー** — 拡張機能アイコンをクリックするだけで、デフォルト形式で即座にコピー
- **4 つのコピー形式**
  - `タイトル + URL` — `Example Page https://example.com`
  - `タイトル + 改行 + URL` — タイトルと URL を改行で区切る
  - `Markdown` — `[Example Page](https://example.com)`
  - `URLのみ` — `https://example.com`
- **クエリパラメータ除去** — URL から `?key=value` 部分を除去してコピー（設定は記憶される）
- **デフォルト形式の変更** — よく使う形式をデフォルトに設定可能
- **コピー状態の表示** — 現在クリップボードにある形式がひと目でわかる

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `clippy-link` フォルダを選択

## 使い方

1. コピーしたいページで拡張機能アイコンをクリック
2. デフォルト形式で自動的にコピーされる
3. 別の形式が必要な場合はボタンをクリックして切り替え
4. 「この形式を次回からのデフォルトにする」にチェックを入れるとデフォルト形式を変更可能

## 技術仕様

- Manifest V3
- 権限: `activeTab`、`storage`
- フレームワーク不使用（Vanilla JS）
